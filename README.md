# AI Cover Letter Generator - SaaS Utility

A dynamic SaaS utility built to streamline the job application process. This project demonstrates proficiency in building robust form-based UIs, managing component state, and implementing efficient text-generation logic for professional use cases.

## Tech Stack

* **HTML5**
* **Vanilla CSS3**
* **Vanilla JavaScript (ES6+)**
* **Core Logic:** Template Literals, DOM API, Clipboard API

## Features Completed (Phase 1: Base MVP)

The following mandatory Phase 1 requirements have been successfully completed:

* **UI Architecture:** Built a clean, responsive form interface capturing essential candidate data: Candidate Name, Job Role, Target Company, and Key Skills.
* **State Management:** Implemented efficient form data capture logic, binding user inputs directly to the application state for seamless processing.
* **Template Generation:** Developed a specialized controller function that uses JavaScript Template Literals to dynamically inject user input into a professional, structured cover letter template.
* **UX/UI Polish:** Integrated a dedicated output component for the generated letter, featuring a functional "Copy to Clipboard" utility to improve user workflow.

## How to Use

1. **Input Details:** Fill out the form with your name, the target company, the role, and your key skills.
2. **Generate:** Click the submit button to trigger the template engine.
3. **Copy:** Once the cover letter is rendered, use the "Copy to Clipboard" button to instantly copy the text for your job application.
